# Student Performance Prediction using Machine Learning

## Project Overview
This project predicts student performance based on study hours and attendance using Machine Learning.
It is a beginner-friendly supervised learning project implemented in Python.

## Technologies Used
- Python
- Machine Learning
- Pandas
- NumPy
- Scikit-learn

## Problem Statement
To predict student marks based on:
- Study Hours
- Attendance Percentage

## Machine Learning Approach
- Supervised Learning
- Linear Regression Algorithm

## Features
- Data handling using Pandas
- Train-test split
- Model training using Linear Regression
- Predicting student marks

## Sample Input
Study Hours: 7  
Attendance: 80  

## Output
Predicted Marks: 75

## Conclusion
This project demonstrates the basic workflow of a Machine Learning model including data preparation, training, and prediction.
